Para poder ejecutar el inicio de sesión de google localmente, necesitan un archivo en la carpeta ssl, el cual se llama
cancert.pem, debe copiar y pegarlo en la siguiente dirección:

C:\wamp64\bin\php\php7.4.26\extras\ssl

Aclaro que a donde dice php7.4.26 es la versión de php que estén usando, el mismo wamp y xamp les dice.

Revisen antes de hacer lo commits.
